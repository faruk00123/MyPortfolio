# 👨‍💻 Md. Omar Faruk's Portfolio

A personal portfolio website built with **React.js** and **Tailwind CSS**, deployed on Vercel.

## 🔗 Live Demo

👉 [my-portfolio-psi-brown-26.vercel.app](https://my-portfolio-psi-brown-26.vercel.app/)

---

## 🛠️ Tech Stack

- ⚛️ **React.js** — Frontend Library
- 🎨 **Tailwind CSS** — Styling
- ⚡ **Vite** — Build Tool
- 🌐 **Vercel** — Deployment

---

## ✨ Features

- 📱 Fully Responsive Design
- 🎯 Projects Showcase Section
- 📬 Contact Section
- 🌙 Clean & Modern UI

---

## 🚀 Getting Started Locally

```bash
# Clone the repository
git clone https://github.com/faruk00123/MyPortfolio.git

# Go into the project directory
cd MyPortfolio

# Install dependencies
npm install

# Start the development server
npm run dev
```
